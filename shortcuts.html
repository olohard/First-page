Skróty klawiszowe w Windows 10

Skróty klawiszowe to bardzo przydatny temat dla osoby, która wiąże z komputerem swoją przyszłość. Skróty klawiszowe działające na Windows 10:
  • Ctrl + C   =>   Kopiuj,
  • Ctrl + V   =>   Wklej,
  • Ctrl + X   =>   Wytnj,
  • Ctrl + P   =>   Drukuj
  • Ctrl + Backspace   =>   Usuwanie całego wyrazu lub ciągu znaków po lewej stronie od kursora
  • Ctrl + Tab   =>   (Głównie w przeglądarce) Przełącznie się pomiedzy kartami
  • Alt + Tab   =>   Przełączanie między włączonymi oknami
  • Windows + Tab   =>   ??Panel wirtualnych pulpitów??
  • Windows + Ctrl + ←   =>   Przejście do wirtualnego pulpitu (w lewo)
  • Windows + Ctrl + →   =>   Przejście do wirtualnego pulpitu (w prawo)
  • Shift + ←   LUB   Shift + →   =>   Zaznaczenie jednego znaku (w lewo lub w prawo)
  • Shift + Ctrl + ←   LUB   Shift + Ctrl + →   =>   Zaznaczenie jednego wyrazu lub ciągu znaków (w lewo lub w prawo)
